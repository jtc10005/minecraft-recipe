## Usage

1. Select which mods you want to enable using the checkboxes
2. Search for an item in the dropdown
3. Click "Show Crafting Tree" to see the complete crafting chain
4. View the raw materials summary at the bottom

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build tools required
- Modular recipe system

## Deployment

This site is deployed on Vercel and can be accessed at your deployment URL.

## Local Development

Simply open `index.html` in a web browser, or run a local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`
